A

  • Atributo: Informação adicional em uma tag HTML que modifica seu comportamento. Exemplo: Link.

  • Alt (atributo): Define um texto alternativo para imagens, útil para acessibilidade e SEO. Exemplo: <img src="imagem.jpg" alt="Descrição da imagem">.

B

  • Body: Elemento <body>, onde fica o conteúdo visível da página HTML.

  • Break Line: Tag <br> usada para quebrar linha dentro de um texto.

C

  • CSS (Cascading Style Sheets): Linguagem usada para estilizar elementos HTML.

  • Class (atributo): Define uma ou mais classes para um elemento, útil para estilização com CSS e manipulação com JavaScript. Exemplo: <div class="caixa">.

D

  • DOCTYPE: Declaração que informa ao navegador a versão do HTML usada no documento.

  • Div: Elemento <div> usado para agrupar e estruturar conteúdo.

E

  • Elemento: Estrutura completa de uma tag HTML, incluindo abertura, conteúdo e fechamento, se aplicável.

  • Em (ênfase): Tag <em> usada para destacar um texto em itálico.

F

  • Formulário: Conjunto de elementos HTML (<form>, <input>, <button>) usados para entrada de dados do usuário.

G

  • GIF (Graphics Interchange Format): Formato de imagem utilizado para imagens animadas.

  • Grid: Sistema de layout utilizado no CSS para criar layouts com múltiplas colunas e linhas.

H

  • Head: Seção <head> do HTML que contém metadados, links para CSS, scripts e título da página.

  • Header: Elemento <header> usado para cabeçalhos de páginas ou seções.

  • Hiperlink: Criado com a tag <a>, permite navegar para outras páginas.

I

  • ID (atributo): Identificador único para um elemento HTML.

  • Iframe: Elemento <iframe> usado para incorporar páginas dentro de outras páginas.

  • Imagem: Elemento <img> usado para exibir imagens.

J

  • JavaScript: Linguagem de programação utilizada para adicionar interatividade e comportamento dinâmico nas páginas web.

  • JSON (JavaScript Object Notation): Formato de dados leve e fácil de ler e escrever, usado para trocar informações entre cliente e servidor.

K

  • Keyword (Palavra-chave): Termo utilizado para descrever a funcionalidade de algo, como palavras-chave no SEO ou no código.

  • Kiosk: Aplicativo web de tela cheia, muitas vezes usado em displays públicos ou para interatividade com o usuário.

L

  • Lista Ordenada: Criada com <ol>, exibe uma lista numerada.

  • Lista Não Ordenada: Criada com <ul>, exibe uma lista com marcadores.

M

  • Meta: Elemento <meta> usado para metadados, como codificação e SEO.

N

  • Nav: Elemento <nav> usado para menus de navegação.

O

  • Objeto (JavaScript): Estrutura de dados que contém propriedades e métodos.

  • Opacidade: Propriedade CSS que define a transparência de um elemento.

P

  • Parágrafo: Tag <p> usada para estruturar textos.

  • Placeholder: Atributo usado em <input> para exibir um texto de dica dentro do campo.

Q

  • Query (Consulta): Referência a uma solicitação de dados ou a busca de informações em bancos de dados, frequentemente usada em consultas SQL ou CSS.

R

  • Responsivo: Técnica de design de sites que se adaptam a diferentes tamanhos de tela, como dispositivos móveis e desktops.

S

  • Section: Elemento <section> usado para dividir seções do conteúdo.

  • Span: Elemento <span> usado para estilizar partes específicas do texto.

T

  • Tabela: Criada com <table>, <tr> (linha), <td> (célula).

  • Title: Define o título da página (exibido na aba do navegador).

  • Tags: Marcadores HTML, como <h1>, <p>, <a>.

U

  • URL (Uniform Resource Locator): Endereço de um recurso na web.

V

  • Variável (JavaScript): Espaço de armazenamento temporário para dados que podem ser alterados durante a execução do código.

  • Viewport: Área visível da página web em dispositivos móveis.

W

  • Web API: Conjunto de interfaces programáticas fornecidas por navegadores para interagir com funcionalidades específicas do sistema.

  • WebSocket: Protocolo que permite comunicação bidirecional em tempo real entre cliente e servidor.

X

  • XML (Extensible Markup Language): Linguagem de marcação que define regras para documentos estruturados, similar ao HTML, mas mais flexível.

  • XHTML: Versão mais rigorosa e XML do HTML.

Y

  • Yaml: Linguagem de serialização de dados usada para arquivos de configuração, especialmente em sistemas web.

  • YouTube API: Interface que permite a integração de vídeos do YouTube em sites, proporcionando funcionalidades personalizadas.

Z

  • Zero Downtime Deployment: Técnica de implantação de software onde as atualizações ocorrem sem interrupção do serviço.

  • Zona (no contexto de CSS): Seções definidas para layout ou posição de elementos na página.
