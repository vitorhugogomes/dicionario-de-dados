Dicionário de Dados - TPA

ministrado pelo professor Rafael

A

  • Algoritmo - Sequência de passos para resolver um problema;

  • Argumento - Valor passado para uma função ou procedimento;

B

  • Booleano - Tipo de dado que assume verdadeiro ou falso;

  • Break - Comando que encerra uma estrutura de repetição;

C

  • C++ - Extensão da linguagem C, uma linguagem de baixo nível, de back-end e altamente tipada, ótima para criação de algoritmos lógicos, com diversas aplicações práticas;

  • Condicional - Estrutura que executa comandos com base em uma condição;

  • Compilador - Programa que traduz código fonte para linguagem de máquina;

D

  • Depuração (Debug) - Processo de identificar e corrigir erros no código;

  • Dados - Informações manipuladas por programas;

E

  • Expressão - Combinação de variáveis, operadores e valores que resulta em um valor;

  • Estrutura de dados - Forma de organizar e armazenar dados em um programa;

F

  • Função - Bloco de código que executa uma tarefa específica;

  • Fluxograma - Representação gráfica de um algoritmo;

G

  • Grafo - Estrutura que representa conexões entre objetos;

  • Garbage Collector - Mecanismo que libera memória automaticamente;

H

  • Heap - Área de memória usada para alocação dinâmica;

  • Herança - Mecanismo da programação orientada a objetos para reutilização de código;

I

  • Iteração - Repetição de um conjunto de instruções;

  • Interface - Contrato que define como uma classe deve se comportar;

J

  • Java - Linguagem de programação orientada a objetos e multiplataforma;

  • JSON - Formato leve de troca de dados baseado em texto;

K

  • Key (Chave) - Identificador único em estruturas de dados como dicionários;

  • Keyword - Palavra reservada da linguagem de programação;

L

  • Laço de repetição - Estrutura que repete um bloco de código;

  • Linguagem de programação - Linguagem usada para escrever algoritmos;

M

  • Método - Função associada a um objeto ou classe;

  • Matriz - Estrutura de dados bidimensional;

N

  • Nó - Elemento em uma estrutura de dados como listas ou árvores;

  • Null - Valor nulo ou inexistente;

O

  • Operador - Símbolo que realiza operações sobre dados;

  • Objeto - Instância de uma classe com atributos e comportamentos;

P

  • Parâmetro - Valor usado para personalizar a execução de uma função;

  • Pilha (Stack) - Estrutura de dados do tipo LIFO (last in, first out);

Q

  • Queue (Fila) - Estrutura de dados do tipo FIFO (first in, first out);

  • Query - Comando usado para buscar dados, geralmente em bancos de dados;

R

  • Recursão - Função que chama a si mesma para resolver subproblemas;

  • Return - Instrução que encerra uma função e devolve um valor;

S

  • Sintaxe - Regras que definem a estrutura válida de comandos na linguagem;

  • Switch - Estrutura condicional de múltiplas escolhas;

T

  • Tipo de dado - Categoria que define o formato e comportamento dos dados;

  • Token - Unidade básica de código analisada pelo compilador;

U

  • Unicode - Padrão de codificação que representa texto em sistemas computacionais;

  • URL - Endereço que localiza um recurso na internet;

V

  • Variável - Espaço na memória que armazena um valor mutável;

  • Vetor - Estrutura de dados unidimensional;

W

  • While - Laço de repetição que executa enquanto a condição for verdadeira;

  • Wrapper - Objeto que encapsula um tipo primitivo em programação orientada a objetos;

X

  • XML - Linguagem de marcação para armazenamento e transporte de dados;

  • XOR - Operação lógica que retorna verdadeiro se os operandos forem diferentes;

Y

  • Yield - Comando que pausa a execução de uma função geradora;

  • YAML - Linguagem de serialização de dados legível por humanos;

Z

  • Zero-based index - Índice de estruturas que começa em zero;

  • Zombie process - Processo finalizado que ainda permanece na tabela de processos;
