Software

A

  • API (Application Programming Interface): Conjunto de regras que permite que programas se comuniquem entre si.

  • Antivirus: Software criado para detectar, impedir e remover programas maliciosos (vírus).

  • Atualização (Update): Processo de instalar melhorias ou correções em um software ou sistema.

B

  • Backup: Cópia de segurança de dados para evitar perda em caso de falhas.

  • Boot: Processo de inicialização do sistema operacional ao ligar o computador.

C

  • Compilador: Programa que converte código-fonte em código executável.

  • Cloud Computing (Computação em Nuvem): Tecnologia que permite acessar arquivos e programas via internet.

D

  • DNS (Domain Name System): Sistema que traduz nomes de domínio em endereços IP.

  • Driver: Programa que permite que o sistema operacional se comunique com o hardware.

  • Download: Transferência de arquivos da internet para o computador.

E

  • Editor de texto: Programa usado para criar e editar arquivos de texto (ex: Notepad, Word).

  • E-mail: Sistema de envio e recebimento de mensagens eletrônicas.

F

  • Firewall: Sistema de segurança que monitora e controla o tráfego de rede.

  • Firmware: Software integrado ao hardware, geralmente imutável ou pouco modificável.

  • Formatação: Processo de apagar e reorganizar dados de uma unidade de armazenamento.

G

  • Gateway: Dispositivo que conecta redes diferentes e converte protocolos.

H

  • Hyperlink: Elemento clicável que leva a outra página ou documento.

I

  • Ícone: Representação gráfica de um programa ou função.

  • Interface gráfica (GUI): Meio de interação entre o usuário e o sistema ou entre sistemas.

J

  • Java: Linguagem de programação orientada a objetos, muito usada em aplicações web e móveis.

K

  • Kernel: Núcleo do sistema operacional, responsável por gerenciar o hardware e os processos.

L

  • Login: Processo de autenticação para acesso a um sistema ou conta.

  • Loop (Laço): Instrução de repetição em programação.

M

  • Malware: Programa malicioso que pode danificar ou roubar dados.

  • Middleware: Software que faz a comunicação entre diferentes sistemas ou aplicações.

N

  • Navegador: Software usado para acessar páginas da internet (ex: Chrome, Firefox).

O

  • Open Source (Código Aberto): Software cujo código é livre para uso, modificação e distribuição.

  • OCR (Reconhecimento Óptico de Caracteres): Tecnologia que converte imagens em texto digital.

P

  • Patch: Correção ou atualização pontual de software.

Q

  • QoS (Quality of Service): Técnica para garantir qualidade em redes, priorizando tráfego.

  • Query: Consulta feita a um banco de dados.

  • Query SQL: Comando utilizado para acessar ou manipular dados em bancos de dados relacionais.

  • Queue (Fila): Estrutura de dados que armazena informações em ordem sequencial.

R

  • RAR: Formato de compressão de arquivos.

S

  • Script: Conjunto de instruções de programação executadas de forma automática.

  • Sistema de Arquivos (File System): Estrutura que organiza e armazena dados em dispositivos de armazenamento.

  • Sistema Operacional (SO): Software fundamental que gerencia hardware e software do computador.

  • Spyware: Tipo de malware que coleta informações do usuário sem permissão.

T

  • Thread: Linha de execução independente dentro de um processo.

  • Token: Código ou dispositivo usado para autenticação de segurança.

U

  • URL (Uniform Resource Locator): Endereço de um recurso na internet.

V

  • Versão: Identificação de uma edição específica de software.

  • VPN (Virtual Private Network): Rede privada virtual que garante segurança e anonimato.

W

  • Web Service: Serviço acessado via internet que permite a troca de dados entre sistemas.

  • Widget: Componente gráfico interativo de uma interface, como relógios ou botões.

X

  • XAML (eXtensible Application Markup Language): Linguagem para definição de interfaces gráficas em aplicações Windows.

  • XHTML (eXtensible Hypertext Markup Language): Variante mais rígida do HTML, com regras mais estritas.

  • XML (eXtensible Markup Language): Linguagem de marcação usada para estruturar dados.

Y

  • YAML (Yet Another Markup Language): Linguagem de marcação leve e legível para configuração de dados.

  • Y-axis: Eixo vertical usado em gráficos e sistemas de coordenadas.

Z

  • Zero-day: Vulnerabilidade de software desconhecida pelo fabricante, explorada por hackers.

  • ZIP: Formato de compressão de arquivos.

Hardware

A

  • AGP (Accelerated Graphics Port): Antigo sistema de conexão para placas de vídeo.

  • AMR (Audio/Modem Riser): Slot de expansão para conectar placas de modem, rede ou áudio.

  • Arquitetura Computacional: Conjunto de design e especificações que definem o hardware.

B

  • Barramento (BUS): sistema de comunicação que permite a troca de dados entre os componentes de um computador, como CPU, memória e dispositivos periféricos.

  • BIOS (Basic Input/Output System)/UEFI: Firmware essencial que inicia o hardware antes do carregamento do sistema operacional.

  • Bit: Unidade mínima de informação digital, podendo ser 0 ou 1.

  • Buffer: Pequena quantidade de memória que armazena dados temporariamente.

C

  • Cache: Memória de acesso rápido que armazena dados temporários para acelerar o sistema.

  • Cooler/Heatsink: O "cérebro" do computador, responsável por processar instruções.

  • Computador: Dispositivo eletrônico que processa informações e executa programas.

D

  • Daemon: Um processo em segundo plano que se encarrega de tarefas específicas num sistema operacional, especialmente em servidores.

  • Debugging: O processo de identificar e corrigir erros ou "bugs" em software ou hardware para melhorar seu desempenho.

E

  • Ethernet: Tecnologia usada para redes locais cabeadas.

  • Extranet: Rede que permite o acesso externo limitado aos recursos internos de uma organização.

F

  • Fonte de alimentação: Componente que fornece energia elétrica para o computador.

G

  • Gigabyte (GB): Unidade de medida de armazenamento digital, equivalente a cerca de 1 bilhão de bytes.

  • Gigahertz (GHz): Unidade de medida de velocidade de processamento.

H

  • HD/SSD:

  • Hostname: Nome atribuído a um computador em uma rede.

I

  • IP (Internet Protocol): Endereço numérico que identifica dispositivos na rede.

J

  • Joystick: Dispositivo de entrada usado para controlar jogos.

  • Jumper: Pequeno conector usado em placas para configurar funções do hardware.

K

  • Kernel: Núcleo do sistema operacional, responsável por gerenciar o hardware e os processos.

  • Keyboard (Teclado): Dispositivo de entrada para digitação de dados.

  • Kit Multimídia: Conjunto de componentes (alto-falante, microfone, etc.) usados para som e imagem.

  • Kbps (Kilobits por segundo): Medida de velocidade de transmissão de dados.

L

  • LAN (Local Area Network): Rede local que conecta computadores próximos entre si.

M

  • Memória RAM (Random Access Memory): Memória volátil usada para armazenar dados temporários durante a execução de programas.

  • Modem: Dispositivo que converte sinais digitais e analógicos para acesso à internet.

N

  • Notebook (Laptop): Computador portátil.

  • Núcleo (Core): Unidade de processamento dentro de uma CPU.

O

  • Overclock: Técnica para forçar um componente a operar acima da velocidade padrão.

P

  • Pendrive (Flashdrive): Dispositivo portátil de armazenamento USB.

  • Placa-mãe (Mother-board): Placa principal que conecta todos os componentes do computador.

  • Placa de vídeo (GPU): Processador gráfico, usado principalmente para vídeos e jogos.

  • Pixel: Menor unidade de uma imagem digital.

  • Processador (CPU):

Q

  • Quibit: Abreviatura de bit quântico, a unidade básica da informação na computação quântica. Os cubits são análogos aos bits nos computadores convencionais.

R

  • Rede (Web): Conjunto de dispositivos conectados que compartilham dados.

  • Resolução: Número de pixels exibidos em uma tela, define a qualidade da imagem.

  • Roteador: Dispositivo que distribui o sinal de internet para vários aparelhos.

S

  • SSD (Solid State Drive): Unidade de armazenamento mais rápida que o HD convencional.

T

  • Tela: Dispositivo de saída de vídeo, também chamada de monitor.

U

  • Unidade lógica: Representação de um dispositivo de armazenamento (ex: C:).

  • USB (Universal Serial Bus): Padrão de conexão para periféricos.

V

  • Vídeo onboard/offboard: Tipo de placa de vídeo integrada ou dedicada ao sistema.

  • Virtualização: Técnica que permite criar versões simuladas de hardware ou sistemas.

W

  • Wi-Fi: Tecnologia de conexão sem fio.

  • Webcam: Câmera usada para capturar vídeo, geralmente conectada ao computador.

X

  • X86 / X64: Arquiteturas de processadores (32 bits e 64 bits).

Y

  • Yottabyte: Unidade de medida de dados muito grande (1 trilhão de terabytes).

Z

  • Zettabyte: Unidade de medida de dados, maior que o exabyte e menor que o yottabyte.

  • Zip Drive: Dispositivo antigo usado para armazenar dados em discos removíveis.
