Dicionário de Dados - DD

ministrado pelo professor Rodrigo Albino

A

  • Acessibilidade: Princípios de design que garantem que produtos digitais sejam utilizáveis por pessoas com deficiências.

  • Alinhamento: Organização dos elementos visuais para criar ordem e clareza.

B

  • Branding: Identidade visual e estratégica de uma marca (logo, cores, estilo).

  • Briefing: Documento com informações essenciais para iniciar um projeto de design.

C

  • Contrast (Contraste): Diferença entre elementos visuais para destacar e hierarquizar informações.

D

  • Dashboard: Interface visual com métricas e informações.

E

  • Espaçamento: Gestão do espaço entre elementos em uma composição.

  • Estilo Visual: Conjunto de características visuais que definem uma identidade.

F

  • Flat Design: Estilo de design minimalista e sem efeitos tridimensionais.

  • Fonte: Tipo de letra utilizada em textos.

G

  • Grid: Estrutura em forma de grade usada para alinhar e organizar elementos no design.

  • Gestalt: Teoria da psicologia que explica como percebemos padrões visuais.

H

  • Hierarquia Visual: Organização dos elementos para guiar o olhar do usuário.

  • Hover: Efeito aplicado quando o mouse passa sobre um elemento.

I

  • Identidade Visual: Conjunto de elementos gráficos que representam uma marca.

J

  • JPEG: Formato de imagem comprimido, comum na web.

  • Justificado: Alinhamento de texto com bordas retas nos dois lados.

K

  • Kerning: Ajuste do espaçamento entre letras.

  • Key Visual: Imagem principal usada para representar campanhas ou marcas.

L

  • Landing Page: Página focada em conversão, usada para campanhas de marketing e captação.

  • Layout: Distribuição organizada dos elementos visuais em uma página ou tela.

  • Logotipo: Representação gráfica do nome da marca.

M

  • Margem: Espaço ao redor de elementos para separar e organizar.

  • Mockup: Representação visual realista de um design, usada para apresentação.

N

  • Navegabilidade: Facilidade com que o usuário encontra e interage com conteúdos em um site ou app.

  • Negative Space: Espaço vazio usado para destacar elementos importantes.

O

  • Overlay: Camada visual que aparece sobre o conteúdo principal.

  • Opacidade: Nível de transparência de um elemento visual.

P

  • Pixel: Menor unidade de uma imagem digital, base da resolução visual.

  • Prototipagem: Criação de modelos interativos para testar funcionalidades e navegação.

Q

  • Quebra de Linha: Separação do texto em linhas distintas.

R

  • Resolução: Quantidade de pixels exibidos na largura e altura de uma imagem ou tela.

  • Responsive Design: Técnica de design que adapta a interface a diferentes tamanhos de tela.

  • RGB: Modelo de cores baseado em vermelho, verde e azul.

  • Responsividade: Capacidade do design de se adaptar a diferentes dispositivos.

S

  • Saturação: Intensidade das cores.

  • Sombra: Efeito visual que dá profundidade a elementos.

  • SVG (Scalable Vector Graphics): Formato de imagem vetorial que permite escalabilidade sem perda de qualidade.

T

  • Template: Modelo reutilizável de layout ou estrutura visual.

  • Tipografia: Estudo e uso das fontes e estilos de texto para comunicar visualmente.

U

  • UX (User Experience): Experiência geral do usuário ao interagir com um produto digital.

  • UI (User Interface): Interface gráfica com a qual o usuário interage em sites e apps.

V

  • Vector: Imagem baseada em fórmulas matemáticas, redimensionável sem perda de qualidade.

  • Vetorização: Processo de converter imagens em gráficos vetoriais escaláveis.

  • Visual Design: Planejamento dos aspectos visuais de uma interface.

W

  • Wireframe: Esboço estrutural de uma interface digital.

  • Whitespace: Espaço em branco que organiza e dá respiro ao design.

X

  • XML: Linguagem de marcação usada para organizar e armazenar dados.

  • XP (eXperience Points): Em design de jogos, indica progresso do usuário.

Y

  • Yellow (Amarelo): Cor usada frequentemente para chamar atenção.

  • Y-Axis: Eixo vertical usado para medir dados visuais em gráficos.

Z

  • Zoom: Aumento ou redução da visualização de um elemento.

  • Z-Index: Propriedade CSS que define a ordem dos elementos sobrepostos.
